## ✨ Features

- 🎵 **Offline playback** — plays music directly from your device storage and SD card
- 🎨 **Dynamic UI** — album art colors paint the Now Playing screen
- 📋 **Playlists** — create, edit and manage custom playlists
- 🔁 **Repeat modes** — Off, Repeat All, Repeat One (like Spotify)
- 🔀 **Shuffle** — randomized playback
- 📜 **Queue** — see and jump to any song in the queue
- 🔔 **Lock screen controls** — control playback from the lock screen
- 🔉 **Background playback** — music keeps playing while using other apps
- 🎧 **Beautiful headphones icon** with teal glow

## 📲 Download

Download the latest APK from the [Releases](../../releases) page.

**Installation:**
1. Download `TealWave.apk`
2. On your phone go to **Settings → Security → Install unknown apps → Allow**
3. Open the APK file and tap **Install**

## 🛠️ Build from Source

### Requirements
- Flutter 3.19.0 or higher
- Android SDK 36
- Java 8

### Steps
```bash
git clone https://github.com/YOUR_USERNAME/TealWave.git
cd TealWave
flutter pub get
